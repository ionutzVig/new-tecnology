== Description ==

eCommerce Companion is created for <a href="https://sellerthemes.com/" rel="nofollow">Seller Theme</a>. The plugin set frontpage sections for navigation, info, slider, category, products & blog. It allow edit customizer settings for the theme. Extend your theme functionalities with one click import & enjoy free stock images.

**Features:**
– Setup Header Section
– Set Slider Section
– Set Info Section
– Set Category Section
– Set Product Section
– Set Blog/Post Section
– Setup Footer Section

You can check out our themes demo <a href="https://sellerthemes/themes/" rel="nofollow" target="_blank">Click Here</a> for a more detailed overview of the features.

== Installation ==

= Option 1. =

- Upload the entire `ecommerce-companion` zip file to the `/wp-content/plugins/` directory.
- Activate the plugin through the 'Plugins' menu in WordPress.

= Option 2. =

- From your wp-admin —> Plugins —> Add New —> Search "eCommerce Companion" —> Click install.
- After plugin is downloaded, click active plugin
